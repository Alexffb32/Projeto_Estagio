Projeto de Estágio

O nosso projeto de estágio é desenvolver uma Aplicação WEB para verificar algumas informações sobre um site, nomeadamente, se posui certeficado SSL, qual o seu codigo de resposta , o seu tempo de resposta e se houve alguma alteração de conteudo.

para testar estas funções insire aqui um URL :

Certeficado SSL

Verificar se a URL possui HTTP ou HTTPS.

Alteração de conteúdo

compara o site entre um intervalo de tempo e notifica se houve alguma mudança ou não.
Isto pode ser feito atráves de API

Código de Resposta

Para Verificar se a status é 200, quando se é Client, podemos ver na consola -> Network -> Doc.

Tempo de resposta

Cacular e mostrar o Time, quando se é Client, podemos ver na consola -> Network -> Doc.
